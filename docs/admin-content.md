# Admin Guide: Blog, Vlog, Documentary

This is the editor-facing workflow for adding written posts, YouTube vlogs, and documentary entries.

## Blog Post
1. Go to Content Manager → Blog Posts.
2. Click Create new entry.
3. Fill required fields:
- `title`
- `excerpt`
- `content`
- `authorName`
4. Add `featuredImage` and `gallery` if available.
5. Set `publishedOn` (optional but recommended).
6. Save and Publish.

## Vlog (YouTube)
1. Go to Content Manager → Vlogs.
2. Click Create new entry.
3. Provide either:
- `youtubeUrl` (preferred)
- or `youtubeVideoId`
4. The system will validate the YouTube URL and auto-fill the ID when possible.
5. Add `thumbnail` and `transcript` if available.
6. Save and Publish.

## Documentary
1. Go to Content Manager → Documentaries.
2. Click Create new entry.
3. Set `status`, `logline`, and `synopsis`.
4. Add `poster` and trailer fields if available.
5. Save and Publish.

## SEO
- For each entry, attach an `SEO Page Meta` record when possible.
- Titles should be short and keyword-aware.
